Built for API Shield

Every API endpoint is a test surface

Slimeopolis is architected to demonstrate all five Cloudflare API Shield capabilities. Each route group is designed to exercise a specific protection feature — from schema validation on checkout to mTLS on wholesale endpoints.

🔎

Schema Validation

/api/auth, /api/cart, /api/orders, /api/products

🔐

Mutual TLS (mTLS)

/api/wholesale/inventory, /api/wholesale/bulk-order

🔗

Sequence Analytics

browse → product → recommendations → cart → checkout

⚡

Rate Limiting + Volumetric Abuse Detection

/api/auth/login, /api/auth/register, /api/products/search
